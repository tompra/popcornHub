Add new user

POST - /users

Description

Add detailed information about a new user.

Parameters

None

Request Data

A JSON object holding data about the user to add. Example:

{ "username": "lizzardWizzard", "password": "123456789", "email": "lizzwizz@mail.com", "birthday": "1990-09-19" }

Response Data

  • Status Code: 200 (OK)
  • Content-Type: JSON
  • Response Body: A JSON object holding data about the user that was added, including an ID. Example: { "username": "lizzardWizzard", "password": "$2b$10$48Vs8X2WU814ycsudswfve2YJKhtSnLN8vJs8ECsleV3rUpLoaMTK", "email": "lizzwizz@mail.com", "birthday": "1990-10-09T23:00:00.000Z", "favoriteMovies": [], "_id": "6526c538edaf653bb841b42a", "__v": 0 }

Update user information

PUT - /users/{username}

Description

Update information from a user.

Parameters

'username' - (required)

Request Data

A JSON holding the data from the user to add. Example:

{ "username": "dragon123", "password": "12345678", "email": "dragon123@mail.com", "birthday": "11-01-2003" }

Response Data

  • Status Code: 200 (OK)
  • Content-Type: JSON
  • Response Body: A message indicating the user's, username, password, email and date of birth was updated. Example: { "_id": "6526bde801b138a6c032067f", "username": "dragon123", "password": "12345678", "email": "dragon123@mail.com", "birthday": "2003-10-31T23:00:00.000Z", "favoriteMovies": [], "__v": 0 }
