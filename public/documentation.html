Welcome to the PopcornHub documentation. This guid will help you intereact with our movie-related data.

Request URL HTTP Method Request Data Format Response Data Body Format
All movies '/movies' GET None A JSON object holding data of all movies
Description of a specific movie '/movies/:title' GET None A JSON object holding data about a single movie by title
Description of a movie by genre's name 'movies/genre/:genreName' GET None A JSON object holding data about a genre by title
Description of a director's name 'movies/directors/:directorsName' GET None A JSON object holding data about the directors bio
Add a new user '/users' POST A JSON object holding data about the user to add A JSON object holding data about the user that was added, including an ID
Update user's info '/users/:id' PUT None A message indicating the user's, username, password, email and date of birth was updated.
Users add movie to favorite list 'users/:id/:addFavoriteTitle' POST A JSON object holding data about the movie to add into the user's favorite list A JSON object holding data about the movie that was added, including an ID
Users delete movie from favorite list 'users/:id/:removeFavoriteTitle' DELETE None A message indicating the user's favorite movie was deleted
Users deregistration 'users/:id' DELETE None A message indicating the user's deregistration
